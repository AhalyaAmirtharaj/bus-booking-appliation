# Bus Booking Application

## 🚀 Project Description
A full-stack Bus Booking System developed for HCL Hackathon.

## 🛠 Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React (Vite)
- Database: MySQL

## ✨ Features
- User Registration & Login
- Bus Search
- Seat Selection
- Ticket Booking
- View My Tickets

## 📁 Project Structure
/backend  
/frontend

## ⚙️ How to Run
1. Start backend (Spring Boot)
2. Start frontend (npm run dev)
3. Open browser
